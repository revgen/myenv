### Change lockscreen background image
1. Open ```/Library/Caches/Desktop Pictures/``` directory
2. Open directory which is look like ```xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx```
    If you have more then one ```xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx``` directories, you need to open your user specific directory (check directory owner).
3. Backup an original lockscreen.png
4. Overwrite lockscreen.png image with your new image
